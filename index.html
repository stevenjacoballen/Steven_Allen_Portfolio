Machine Learning

Predicting Flight Delays

This project attempts to determine how accurately Delta Airlines flight delays can be predicted using airline performance and weather data. Recall will serve as the primary accuracy metric.

Outline

  1. Import and Clean Data: Two datasets consisting of airline performance and weather data.
  2. PostgreSQL Database: Create database from the cleaned datasets above, then query database for finalized project dataset. Exploratory Data Analysis.
  3. Machine Learning: Scale data and train the two following models for binary classification (delayed flight or not delayed) to establish baseline performance. Both models will also undergo hyperparameter tuning and will have run-times and performance tracked.
      • Logistic Regression
      • Gradient Boosting Classifier
  4. Principal Component Analysis: Conduct PCA on data set to reduce dimensionality and ideally, increase model performance/efficiency.
  5. Retrain Models: Retrain both models after PCA using the optimally tuned hyperparameters.
  6. Results, Observations, Conclusion: Evaluate baseline, tuned, and post-PCA performance for both ML models.

Tableau

Delta Airlines Flight Performance

This dashboard features five maps at the top, one for each of Delta Airlines’ busiest airports. These maps display each airport’s top N destinations within a selected date range.

The main body of the dashboard consists of charts and KPI data points that visualize the aggregate performance of the top five airports within a selected date range. The use of average lines and conditional formatting assisst the user in quickly identifying potential problems within the operation.

This dashboard was created from the finalized dataset from the Predicting Flight Delays project (above).

COVID Tracker

This interactive COVID Tracker dashboard allows the user to view several high-level data points about a selected country, in addition to charts that visually compare different aspects of the data set, such as: monthly deaths compared to monthly cases, monthly deaths compared to the number of people getting fully vaccinated each month, and more.

This dashboard was created from splitting a dataset that was gathered from https://ourworldindata.org/covid-deaths. The datasets were imported Microsoft SQL Server Management Studio as two separate tables, where several views were created from SQL queries. These views were then imported into Tableau to serve as the data sources for the different components of the Tableau dashboard.
