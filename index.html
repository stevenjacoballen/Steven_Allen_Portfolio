Tableau

Delta Flight Performance

Created from the finalized dataset from the Flight Delays project found in the Machine Learning tab above, this dashboard features five maps at the top, one for each of Delta Airlines’ busiest airports. These maps display each airport’s top N destinations within a specific date range. Top N destinations and the date range can be modified by the user.

The main body of the dashboard consists of charts and KPI data points that visualize the aggregate performance of the top five airports within the selected date range. The use of average lines and conditional formatting assisst the user in quickly identifying potential problems within the operation.

Tableau + Python

Automation

This project leverages Python scripting to automate the process of updating and publishing a Tableau dashboard with temperature data scraped from a web page every hour.

A brief video can also be viewed, showing the final stage of the script's execution - updating Tableau with PyAutoGUI. All mouse movements and interactions with the UI were completely automated.

Script Process & Libraries Used

  1. Hourly execution of script (crontab)
  2. Request DarkSky.net website (Requests)
  3. Scrape current temperature (Beautiful Soup)
  4. Write data to Excel file (OpenPyXL)
  5. Update Tableau & publish (PyAutoGUI)
    • Open Tableau
    • Refresh data connection
    • Save/publish to Tableau Public
    • Close web browser and Tableau
