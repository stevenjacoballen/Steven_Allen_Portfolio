Tableau

Delta Airlines Flight Performance

Created from the finalized dataset from "Flight Delays" project above, this dashboard features five maps at the top, one for each of Delta Airlines’ busiest airports. These maps display each airport’s top N destinations within a specific date range. Top N destinations and the date range can both be modified by the user.

The main body of the dashboard consists of charts and KPI data points that visualize the aggregate performance of the top five airports within the selected date range. The use of average lines and conditional formatting assisst the user in quickly identifying potential problems within the operation.
